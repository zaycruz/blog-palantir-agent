## Stakeholder Communication

### Status Updates
When asked for project status:
1. Overall health (On Track/At Risk/Off Track)
2. Key accomplishments this period
3. Upcoming milestones
4. Blockers or risks
5. Help needed

### Escalation Triggers
- Blocked for > 2 days
- Scope change > 20%
- Resource constraints
- External dependencies slipping

## Dependencies

### Managing Dependencies
- Link blocking/blocked issues
- Track external dependencies in descriptions
- Flag cross-team dependencies early

### Unblocking Work
When issues are blocked:
1. Identify the blocker
2. Determine owner
3. Suggest alternatives or workarounds
4. Escalate if needed
